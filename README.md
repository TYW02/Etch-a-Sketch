🎨 Etch-a-Sketch (DOM Manipulation Project)
A browser-based Etch-a-Sketch application built with HTML, CSS, and JavaScript, created as part of The Odin Project curriculum. This project focuses on strengthening core DOM manipulation skills and interactive UI handling.

📌 Overview
This project replicates the classic Etch-a-Sketch drawing experience in a web environment. Users can dynamically generate a grid and draw by hovering over squares, demonstrating real-time DOM updates and event handling.

🚀 Features
- 🧱 Dynamic Grid Generation
Create a customizable grid size using user input.
- 🎯 Interactive Drawing
Hover over grid cells to draw.
- 🔄 Reset Functionality
Clear the grid and start fresh with a single click.
- ⚡ Efficient DOM Manipulation
Elements are generated and updated dynamically using JavaScript.

🛠️ Technologies Used
- HTML5 – Structure of the application
- CSS3 – Styling and layout
- JavaScript (Vanilla JS) – DOM manipulation and interactivity

🧠 What I Learned
- Creating and managing elements dynamically in the DOM
- Handling user input and events (click, hover)
- Structuring interactive UI components
- Writing cleaner and more modular JavaScript
